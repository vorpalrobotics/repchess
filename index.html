    Import Variations

    Paste one or more variations in algebraic notation, one variation per line, e.g. 1. e4 e5 2. Nf3 Nc6 3. Bb5 a6. Sets each of your moves as the standard response along each path in the currently open opening system.

    Search for a Variation

    Paste a move sequence starting from move 1, e.g. 1. e4 e5 2. Nf3 Nc6 3. Bb5 a6. If found, expands and focuses on that exact path in the tree; everything else collapses out of view.

Each file is filed into the matching square's mnemonic automatically; existing images for that square/piece are overwritten.
